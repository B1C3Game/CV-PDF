## Projects

### Fraga: Multi-Agent Cognitive Alignment System

Local multi-agent architecture grounded in SOUL (vocabulary-bound identity) and Aishna (shared narrative memory). Routing evaluation: 8/8 scenarios correct (F1=1.00). Recall improvement: +0.737 mean top score. Conference synthesis: 5/5 integrity checks pass. Fully local, fully instrumented, fully reproducible.

[github.com/B1C3Game/fraga](https://github.com/B1C3Game/fraga)

Impact: Demonstrates that constraint-forced semantic framing plus shared memory improve operational legibility and measurable routing behavior.

### Anthropocentric Modulation (B1C3 Clock)

Clock system representing time as humans experience it, not arbitrary divisions. Based on four cognitive axioms: Day Cycle (24h), Waking Arc (16h), Bout (4h), Recovery (30m). Includes fermentation-cycle anchor for multi-day processes.

Features: Core engine (pure Python), CLI interface, interactive web UI with gradient visualization.

[github.com/B1C3Game/anthropocentric-modulation](https://github.com/B1C3Game/anthropocentric-modulation)

Impact: Philosophy made operational. Time representation aligned with human cognition, not convention.
